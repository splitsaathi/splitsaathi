# Privacy Policy — SplitSaathi

**Last updated:** [DATE]

SplitSaathi ("we", "our", "app") respects your privacy. Ye Privacy Policy batati hai ki hum kaunsa data collect karte hain aur kaise use karte hain.

> ⚠️ **Action required:** Is document ko apni company details ke saath edit karo, fir ek free hosting (GitHub Pages, Notion, ya termly.io) par publish karo. App Store aur Play Store dono is URL ko **mandatory** maangte hain.

## 1. Data Hum Collect Karte Hain

| Data Type | Kyun Chahiye |
|-----------|--------------|
| Naam, Email, Phone | Account banane aur login ke liye |
| Profile Photo (optional) | Personalization ke liye |
| Phone Contacts (optional) | Friends dhundhne ke liye — sirf aapki permission ke baad |
| Bill aur Group details | App ki core functionality (expense splitting) ke liye |
| UPI ID (optional) | Payments process karne ke liye |
| Push Notification Token | Reminders aur alerts bhejne ke liye |
| Payment transaction data | Razorpay ke through process hota hai — hum card details store nahi karte |

## 2. Hum Data Kaise Use Karte Hain

- Account banane aur authenticate karne ke liye
- Groups aur bills manage karne ke liye
- Balance calculations dikhane ke liye — **sirf aapko wahi bills dikhte hain jisme aap shamil ho**
- Payment processing (Razorpay third-party gateway use karte hain)
- Reminders aur notifications bhejne ke liye

## 3. Data Sharing

Hum aapka personal data kisi ko **bechte nahi** hain. Data sirf in services ke saath share hota hai:
- **Supabase** (database hosting) — encrypted storage
- **Razorpay** (payment processing) — PCI-DSS compliant
- Group members jinke saath aap bills share karte hain (sirf relevant bill details)

## 4. Data Security

- Saara data encrypted hai (in transit aur at rest)
- Row-Level Security (RLS) policies ensure karti hain ki sirf authorized users hi apna data dekh sakein
- Hum industry-standard security practices follow karte hain

## 5. Aapke Rights

- Apna data dekh sakte ho (Account → Profile)
- Apna data edit/delete kar sakte ho
- Account delete karne ke liye [support email] par contact karo
- Contacts permission kabhi bhi phone settings se revoke kar sakte ho

## 6. Children's Privacy

SplitSaathi 13 saal se kam umar ke bacchon ke liye nahi hai.

## 7. Changes to This Policy

Hum is policy ko update kar sakte hain. Major changes hone par app mein notify karenge.

## 8. Contact Us

Questions ke liye: **support@splitsaathi.app** (apna email yahan daalo)
